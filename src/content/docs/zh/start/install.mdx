## 2. 安装  

### Android 🐸  

直接打开 APK 文件并点击“安装”。  

### iOS 🍎  

由于 iOS 不允许轻松地安装非官方应用，并且我没有额外的 $99/年支付给 Apple 购买证书，所以您需要自己进行一些操作来安装。  

**前置要求**  

- 一个免费的 Apple 账户  
- 一台电脑（Windows / macOS）  

目前有多种方式可以安装自签应用：TrollStore、AltStore/SideStore 和 Sideloadly。  

- **TrollStore** 仅支持 iOS 14.0 beta 2 - 16.6.1、16.7 RC (20H18) 和 17.0，但如果您的 iOS 版本符合要求，它是最佳选择。通过 TrollStore 安装的应用将被永久签名，而其他方法需要每周刷新。  
- **AltStore / SideStore** 界面更美观，且刷新应用更方便。  
    - AltStore 通过局域网连接  
    - SideStore 通过 VPN 连接  
- **Sideloadly** 更高级、功能更多，但使用起来稍显复杂，不如 AltStore / SideStore 方便。  

### 安装方法  

#### 方法 1：使用 AltStore / SideStore  

1. 按照 [AltStore](https://altstore.io/) 或 [SideStore](https://sidestore.io/) 的官方指南，在您的 iOS 设备上安装商店。  
2. 在 iOS 设备上下载 KanadeDX.ipa 文件。  
3. 在 AltStore 或 SideStore 中打开 ipa 文件进行安装。  

#### 方法 2：使用 Sideloadly  

1. 按照 [Sideloadly](https://sideloadly.io/) 的官方指南，在电脑上安装 ipa 文件。  

请注意，使用 Sideloadly 安装的应用只能使用 7 天，之后需要再次连接电脑刷新应用，否则应用将无法打开。
